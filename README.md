# 小鹿奶茶 Mellow Tea 菜单展示网站 Demo

这是一个使用 HTML、CSS、JavaScript 制作的展示型菜单网站 Demo。页面以「小鹿奶茶 Mellow Tea」为示例门店，展示奶茶、咖啡、甜点、营业时间、地址和联系方式，适合用于奶茶店、咖啡店、甜品店等本地小店的网站案例展示。

项目不依赖 React、Vue 或其他前端框架，直接打开 `index.html` 即可预览。

## 项目定位

- 展示本地饮品小店的菜单、价格和基础门店信息
- 适合做小店官网、菜单展示页、开业介绍页的静态页面参考
- 页面内容为示例信息，可按实际门店替换
- 保留响应式布局，电脑端和手机端都可以浏览

## 页面内容

- 固定顶部导航栏
- 首页 Hero 区域
- 今日推荐产品卡片
- 可点击切换的菜单价格
- 关于小店介绍
- 营业时间、地址、电话、微信
- 页脚信息
- 手机端汉堡菜单

## 如何修改店名

打开 `index.html`，搜索：

```text
小鹿奶茶 Mellow Tea
```

替换为实际门店名称即可。

如果要修改左上角圆形 Logo 里的字，可以搜索：

```text
鹿
```

改成门店简称，例如 `茶`、`咖`、`甜`。

## 如何修改菜单和价格

菜单数据放在 `script.js` 顶部的 `menuData` 中。

示例：

```js
{
  name: "黑糖珍珠奶茶",
  description: "黑糖珍珠现煮，茶味和奶味都比较稳，默认七分糖。",
  price: "￥16"
}
```

常用修改项：

- `name`：产品名
- `description`：产品介绍
- `price`：产品价格

当前示例产品：

- 黑糖珍珠奶茶 ￥16
- 芋泥波波奶茶 ￥18
- 茉莉奶绿 ￥15
- 生椰拿铁 ￥20
- 抹茶红豆奶盖 ￥19
- 蜂蜜黄油吐司 ￥22

## 如何修改联系方式

打开 `index.html`，搜索并替换以下示例内容：

- `上海市徐汇区桂林路 88 号阳光里 1F-12`
- `021-6688 5200`
- `xiaolu_milk_tea`

电话按钮目前使用：

```html
<a class="btn btn-primary" href="tel:+862166885200">电话咨询</a>
```

如果门店更常用微信或地图链接，可以把按钮链接改成对应页面。

## 如何修改颜色

主要颜色写在 `style.css` 顶部的 `:root` 中。

常用变量：

- `--cream`：页面背景色
- `--coffee`：深咖啡文字色
- `--orange`：按钮和强调色
- `--caramel`：价格和小标题颜色

建议每次只调整少量颜色，保持页面整体风格统一。

## 本地预览

直接双击打开：


也可以使用 VS Code 的 Live Server 打开，修改代码后刷新页面查看效果。

## 部署到 GitHub Pages

1. 将项目上传到 GitHub 仓库。
2. 进入仓库的 `Settings`。
3. 找到 `Pages`。
4. Source 选择 `Deploy from a branch`。
5. Branch 选择 `main`，目录选择 `/`。
6. 保存后等待 GitHub 生成访问链接。

如果已经开启 GitHub Pages，推送新代码后页面会自动更新。

## 部署到 Netlify

拖拽部署：

1. 打开 [Netlify](https://www.netlify.com/)。
2. 登录账号。
3. 进入 `Sites`。
4. 把整个项目文件夹拖入页面。
5. 等待部署完成。

也可以连接 GitHub 仓库进行部署，构建命令留空，发布目录留空或填写 `/`。
